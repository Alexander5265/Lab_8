# Юркин Александр ИТС-1 Лабораторная №8

# Задание 1.

## Задача 10.

### Текст задачи.

Разработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. 
Тема проекта - Музейные экспонаты.

### Алгоритм решения.

1. Разработка класса Museum со свойствами, конструктором и методом ToString().
2. Создание класса MuseumDatabase для работы с базой данных и бинарным файлом.
3. Реализация сохранения и загрузки данных с помощью сериализации.
4. Реализация функций добавления, просмотра и удаления экспонатов.
5. Создание LINQ-запросов для поиска и фильтрации данных.
6. Разработка консольного меню и пользовательского интерфейса.
7. Добавление проверки корректности ввода данных.
8. Реализация обработки ошибок с помощью try-catch.
9. Тестирование работы программы.
