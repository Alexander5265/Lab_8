# Задание 1.

## Задача 10.

### Текст задачи.

азработать консольное приложение с дружественным интерфейсом с возможностью выбора
заданий для работы с «базой данных (БД)», хранящейся в бинарном файле. 
Тема проекта - Музейные экспонаты.

### Алгоритм решения.

1. Для начала создадим 3 разных класса.
2. Cоздаем класс museum, который содержит в себе свойства объектов, конструкторы и метод ToString.
3. Далее создаем вспомогательный класс museumDatabase, в котором будут содержаться LINQ-запросы и методы сохранения/загрузки.
4. Создаем главный класс program, он содержит сам интерфейс (через switch case) и меню (через Try Catch). Также добавляем меню запросов от пользователя также через switch case.
5. Запускаем программу, добавляем экспонаты и проверяем целостность работы.
